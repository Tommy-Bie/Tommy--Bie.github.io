Xshell+WinSCP实现服务器远程连接

在得到老师实验室服务器分配给自己的账号之后，可以通过Xshell远程连接实验室服务器，我们就可以在自己的电脑上使用实验室显卡进行深度学习的训练啦。服务器的文件使用WinSCP进行管理。

百度下载Xshell、WinSCP

运行Xshell，创建会话->输入会话名称（自己命名）、主机IP, 在SSH窗口中输入用户名和密码之后即可进入命令行界面。

运行WinSCP，输入主机IP、用户名和密码等信息即可，该软件可用于Windows和服务器（Ubuntu）之间的文件传输以及文件管理。
